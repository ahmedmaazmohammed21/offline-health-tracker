# HealthPulse - Offline Desktop Health Tracker

## Setup
1. Install Python 3.8+
2. Run: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Open: http://localhost:5000

## Features
- 🔐 Login/Register system
- 📊 Analytics with interactive charts
- ⚖️ BMI Calculator & health suggestions
- 🔥 Calorie tracker with 30-day history
- 🎯 Goal system with completion tracking
- 🔔 Reminder system with browser notifications
- 🏆 Streak tracking & badges
- 📱 Fully responsive design
- 💾 All data stored locally (offline)
