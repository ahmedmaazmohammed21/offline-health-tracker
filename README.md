# 🏥 Offline Desktop Health Tracker

A desktop application to track daily health and productivity without internet.

---

## 🚀 Features
- 📊 Daily health tracking
- 📅 Calendar with progress tracking
- 📈 Charts (Line, Pie, Bar)
- 🔒 Login system

---

## ⚙️ How to Run

1. Clone the repository:
   git clone https://github.com/ahmedmaazmohammed21/offline-health-tracker.git

2. Install requirements:
   pip install -r requirements.txt

3. Run the app:
   python app.py
